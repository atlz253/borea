# create-skill

## Purpose

This skill helps generate new skills for the Nirvana project by asking for the skill name and description, then creating the necessary files.

## Requirements

- Node.js (v14+ recommended)
- npm installed

## Installation

```bash
cd C:/Projects/nirvana/.agents/skills/create-skill
npm install
```

## Usage

```bash
node create-skill.cjs
```

The skill will guide you through the process of creating a new skill.
